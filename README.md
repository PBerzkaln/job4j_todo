# Проект "TODO-list"

## Описание проекта:

Проект представляет собой web-приложение по созданию списка дел. В список можно добавлять задания, а также
просматривать, редактировать и удалять его. По завершению задания, есть возможно поменять его статус на "выполнено".

## Стек технологий:

- **Java 17**
- **Spring Boot 2.7.6**
- **Lombok 1.18.22**
- **Hibernate 5.6.11**
- **PostgreSQL 14**
- **Liquibase 4.15.0**
- **HTML 5**
- **Thymeleaf 2.7.6**
- **BOOTSTRAP 4.4.1**

## Требования к окружению:

- **Java 17**
- **Maven 3.8**
- **PostgresSQL 14**

## Запуск проекта:

``` shell 
create database todo;
```

